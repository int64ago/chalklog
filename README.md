# fancy-log-chalk
[chalk][1] + [fancy-log][2]

[![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url]

### Usage

 `npm i fancy-log-chalk`

```javascript
var log = require('fancy-log-chalk');

log.red('red');
log.bgGreen('bgGreen');
log.blue.bgRed.bold('blue+bgRed+bold');
```
