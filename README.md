# chalklog
A logger based on [chalk](https://github.com/chalk/chalk)

[![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url]

### Usage

 `npm i chalklog`

```javascript
var log = require('chalklog')('scope');

log.red('red');
log.bgGreen('bgGreen');
log.blue.bgRed.bold('blue+bgRed+bold');
```
